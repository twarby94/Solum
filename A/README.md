# Mystic Waves
## Setup & Compilation Instructions
### Requirements
C++ compiler (g++, clang++, or MSVC)

Windows, Linux, or macOS terminal

### Compile Instructions
Using g++:
```
g++ "Mystic Waves.cpp" -o mystic
```
Using clang++:
```
clang++ "Mystic Waves.cpp" -o mystic
```
## Running the Program
```
./mystic
```
You will be prompted:
```
Input t between 1-100
Input x and n between 1-10
```
### Input Requirements
**t** must be between **1 and 100**

**x** and **n** must both be between **1 and 10**
### Example Output
```
Test case 1
Input x and n between 1-10
5 4
Total energy: 0
```
