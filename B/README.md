# CargoCraft Fleet
## Setup & Compilation Instructions
### Requirements
C++ compiler (g++, clang++, or MSVC)

Windows, Linux, or macOS terminal

### Compile Instructions
Using g++:
```
g++ "CargoCraft Fleet.cpp" -o cargocraft
```
Using clang++:
```
clang++ "CargoCraft Fleet.cpp" -o cargocraft
```
## Running the Program
```
./cargocraft
```
You will be prompted:
```
Input t between 1-1000
Input n between 1-10^18
```
### Input Requirements
**t** must be between **1 and 1000**

**n** must be between **1 and 10^18**
### Example Output
```
Input t between 1-1000
1
Test case 1
Input n between 1-10^18
24
Min: 4 Max: 6
```
